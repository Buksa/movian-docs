# Intermediate Plugin Examples

Common patterns for real-world plugins.

## 01-http-json-api
HTTP requests with JSON parsing. Foundation for most video plugins.

## 02-html-parser
Web scraping with HTML parsing using Gumbo.

## 03-sqlite-storage
Persistent storage: SQLite database and key-value store.

## 04-search-provider
Adding custom search to Movian's global search.

## 05-pagination
Handling multi-page content: "Load More" and numbered pages.

## Principles
- DRY: Don't Repeat Yourself
- Error handling
- Reusable helper functions
