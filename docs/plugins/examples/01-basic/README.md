# Basic Plugin Examples

Simple, minimal examples for beginners.

## 01-hello-world
Minimal working plugin. Shows service creation and basic route.

## 02-simple-video
Demonstrates different item types: video, audio, directory.

## 03-all-settings-types
Shows every available setting type: bool, string, int, multi-opt, action, divider, info.

## Principles
- KISS: Keep It Simple, Stupid
- Minimal code with clear comments
- Focus on one concept per example
